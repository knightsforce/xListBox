При $(СПИСОК).xlistbox(СОЗДАТЬ) - создается НОВЫЙ список на основе существующего

ПЕРВЫЙ созданный список - построен на базе данных из оригинальноо списка, ОСТАЛЬНЫЕ - программно

Список 1. Разметка с атрибутами:

  • Категории
  • Исполнители
  • Элемент выключен
  • Заказы

Список 2. Разметка с data-options:

  • Категории
  • Исполнители
  • Элемент выключен
  • Заказы

Список 1. Создан от элемента. Опции:

selectable = true
movable = false
disabled = false
multiselect = true

Список 2. Создан программно от атрибутов. Опции:

selectable = true
movable = false
disabled = false
multiselect = true

Список 3. Создан программно от data-options. Опции:

selectable = false
movable = true
disabled = false
multiselect = true (не играет роли, т.к. selectable = false)

Список 4. Создан программно от data-options. Опции:

selectable = true
movable = true
disabled = true (интерактив отсутствует)
multiselect = true

Список 5. Создан программно от data-options. Опции:

selectable = true
movable = true
disabled = false
multiselect = false
