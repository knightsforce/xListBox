При $(СПИСОК).xlistbox(СОЗДАТЬ) - создается НОВЫЙ список на основе существующего

Список 1. Создан от элемента через атрибуты. Опции:

selectable = true
movable = true (Стрелок нет. Присутствуют только если у списка (ul) и поля списка (li) movable = true одновременно)
disabled = true (интерактив отсутствует)
multiselect = true
  • Категории
  • Исполнители
  • Элемент выключен
  • Заказы

Список 2. Создан от элемента через data-options. Опции:

selectable = true
movable = true
disabled = false
multiselect = false
  • Категории
  • Исполнители
  • Элемент выключен
  • Заказы

Список 3. Создан от элемента через data-options вызовом $(id).xListBox(). Опции:

selectable = true
movable = true
disabled = false
multiselect = false
  • Категории
  • Исполнители
  • Элемент выключен
  • Заказы

Список 4. Создан программно через переданный объект. Опции:

selectable = true
movable = false
disabled = false
multiselect = true
  • Категории
  • Исполнители
  • Элемент выключен
  • Заказы

Список 5. Создан программно через переданный объект. Опции:

selectable = false
movable = true
disabled = false
multiselect = true (не играет роли, т.к. selectable = false)
  • Категории
  • Исполнители
  • Элемент выключен
  • Заказы
